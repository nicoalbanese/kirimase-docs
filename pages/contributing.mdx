## Reporting Bugs and Feature Requests

-   **Bugs**: If you've found a bug, please open an issue in the GitHub issue tracker. Ensure that the bug was not already reported by searching existing issues.
-   **Feature Requests**: If there's a feature you think would improve Kirimase, feel free to open a new feature request issue. Describe the feature, its use case, and how it should be implemented if possible.
