# `add` Command

The `add` command in Kirimase CLI installs and configures essential packages for your Next.js application in seconds.

## Usage

```bash
kirimase add
```

## Description

Running the `add` command initiates a series of interactive prompts, allowing you to select and configure various packages within predefined categories, thereby customizing your development environment to meet your project's specific needs.
